==Description==
This plugin creates a new widget that outputs contact information (URL, address, email, phone) according to the hCard microformat standard.
In addition, each field can be assigned a unique CSS class for styling.
eg:-
<div id="hcard" class="vcard">
<span class="fn n">
    <span class="given-name">GIVEN NAME</span>
  <span class="additional-name">MIDDLE NAME</span>
  <span class="family-name">FAMILY NAME</span>
</span>
<div class="org">ORGANIZATION</div>
<a class="email" href="mailto:">EMAIL ADDRESS</a>
<div class="adr">
  <div class="street-address">STREET ADDRESS</div>
  <span class="locality">CITY</span>
, 
  <span class="region">STATE OR PROVINCE</span>
, 
  <span class="postal-code">ZIP OR POSTAL CODE</span>

  <span class="country-name">COUNTRY NAME</span>

</div>
<div class="tel">TELEPHONE</div>
</div>

== Installation ==
METHOD 1: USING THE WORDPRESS INSTALLER
1. Download hCard Widget for WordPress plugin (hCard_Widget.zip)
2. Login to WordPress and go to Plugins-> Add New -> Upload
3. Click the “Choose File” button and select ‘hCard_Widget.zip” from your hard drive.
4. Click “Install Now”
5. Once the plugin has been installed, click “Activate”.
METHOD 2: FTP
1.  Download hCard Widget for WordPress plugin (hCard_Widget.zip)
2. Extract the contents of the file.
3. Using your favourite FTP client, upload the folder to /wp-content/plugins/ on your server.
4. Login to your WordPress account and go to the Plugins page.
5. Activate the plugin.
